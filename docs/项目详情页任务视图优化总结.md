# 项目详情页任务视图优化总结

## 实现的功能

### ✅ 1. 列表页父子树形结构显示
- **集成TaskTreeList组件**：复用任务页的树形列表组件
- **自动处理父子关系**：基于`parentTaskId`字段自动构建树形结构
- **可展开/收起**：支持点击展开/收起子任务树
- **状态分组**：保持按工作流状态分组显示
- **多层级支持**：支持多级嵌套的父子任务关系

### ✅ 2. 增加看板视图
- **集成KanbanBoard组件**：复用任务页的看板组件
- **拖拽操作**：支持拖拽任务到不同状态列
- **自动状态更新**：拖拽后自动更新任务状态
- **一致性体验**：与任务页保持相同的交互体验

### ✅ 3. 组件复用
- **TaskTreeList**：树形列表组件，支持父子任务显示
- **KanbanBoard**：看板组件，支持拖拽状态管理
- **TaskCreatePanel**：新建任务面板，支持子任务创建
- **统一事件处理**：复用现有的事件处理逻辑

## 技术实现细节

### 代码结构优化
1. **移除冗余代码**：
   - 删除了手风琴分组相关代码（约200行）
   - 移除了重复的状态管理逻辑
   - 简化了任务列表渲染

2. **组件集成**：
   ```vue
   <!-- 列表视图（支持树形结构） -->
   <TaskTreeList
     v-if="taskViewMode === 'list'"
     :tasks="sortedTasks"
     :project-name-map="projectNameMap"
     :tick="tick"
     mode="compact"
     @transition="handleTaskTransition"
     @edit="handleTaskEdit"
     @delete="deletingTaskId = $event"
     @add-subtask="handleAddSubtask"
   />

   <!-- 看板视图 -->
   <KanbanBoard
     v-else
     :tasks="sortedTasks"
     :project-name-map="projectNameMap"
     :tick="tick"
     @move="handleMoveTask"
     @edit="handleTaskEdit"
     @delete="deletingTaskId = $event"
     @add-subtask="handleAddSubtask"
   />
   ```

3. **新增功能**：
   - `taskViewMode`：管理视图模式（list/kanban）
   - `subtaskParentId`：处理子任务创建
   - `projectNameMap`：为组件提供项目名称映射

### 用户体验改进
1. **视图切换**：
   - 新增视图模式切换按钮
   - 保持用户选择的视图状态
   - 流畅的切换体验

2. **父子任务**：
   - 树形结构清晰显示层级关系
   - 支持添加子任务功能
   - 可视化的展开/收起控制

3. **看板操作**：
   - 直观的拖拽界面
   - 实时状态更新
   - 视觉反馈增强

## 使用指南

### 视图切换
在项目详情页的"任务列表"标签页中：
- **列表视图**：显示树形结构，支持父子任务关系
- **看板视图**：拖拽式管理，快速更改任务状态

### 树形结构操作
- **展开/收起**：点击任务前的箭头图标
- **添加子任务**：点击任务的"添加子任务"按钮
- **多层级**：支持多级嵌套，无限层级

### 看板操作
- **拖拽**：直接拖拽任务卡片到目标状态列
- **自动更新**：拖拽后自动更新任务状态
- **视觉反馈**：拖拽过程中高亮显示目标列
